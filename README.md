# building out loud

### an open source learning blog by Olivia Keiter

---

I am not an AI researcher. I don't have a computer science degree. I'm a U.S. Navy veteran turned operations and automation leader who started using AI tools, got frustrated, figured some things out, got frustrated again, figured out more things, and decided to write it all down.

This is that.

*Building Out Loud is a running blog of what I'm learning about AI, how I'm actually using it in real work, and the mistakes I made so maybe you don't have to. No polish. No sponsorships. No "10 prompts that will change your life."*

Just real notes from someone in the middle of it.

---

## who this is for

If you are brand new to AI tools and have no idea where to start, you're in the right place.

If you've been using Claude or ChatGPT for a while but feel like you're leaving something on the table, also the right place.

If you're a senior leader evaluating AI for your team and you want a ground-level perspective that isn't a vendor deck, pull up a chair.

*If you already know everything, I admire the confidence and appreciate constructive input!*

## a note on how I write

I write from experience. If I got something wrong, I'll say so. If something changed since I wrote it, I'll update it or note it.

I use Claude as my primary AI tool. That's not a sponsorship, it's just true. Some posts are specifically about Claude. The concepts usually apply more broadly.
